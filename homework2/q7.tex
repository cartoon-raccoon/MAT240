\textbf{(Q7)}

\begin{proof}
    Since $(A^t)_{ij} = A_{ji}$, we then have $\forall i, j$:
    
    \[
        ((A^t)^t)_{ij} = (A^t)_{ji} = A_{ij}
    \]

    Which implies the original entry in the original matrix.
\end{proof}
